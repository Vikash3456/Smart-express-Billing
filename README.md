# ⚡ Smart Express Billing

> ⏱️ Skip the queue. 📸 Scan. 💳 Pay. 📩 Walk away with an e-bill!

A smart web application designed for busy malls where customers buying just 1–2 items can quickly scan, pay, and exit — no more waiting in long lines!

---

## 📷 How It Works

1️⃣ **Enter Phone Number**  
2️⃣ **Upload Product Image(s)** with Barcode / QR  
3️⃣ **System Scans & Generates a Bill**  
4️⃣ **Pay via Dummy Gateway (Stripe Test Mode)**  
5️⃣ **Get e-Bill via WhatsApp / Download as PDF**

---

## ✨ Features

- 📸 Upload & scan multiple product barcodes
- 🧾 Instant billing with product preview
- 💳 Dummy payment with Stripe (test card only)
- 📄 Auto-generated PDF bill
- 📲 Simulated WhatsApp bill sending
- 🌐 Mobile-optimized UI

---

## 🛠️ Tech Stack

| Layer       | Tech Used             |
|-------------|------------------------|
| Frontend    | HTML, CSS, JavaScript |
| Barcode     | QuaggaJS / jsQR       |
| PDF Bill    | jsPDF                 |
| Payment     | Stripe (Test Mode)    |
| WhatsApp    | `wa.me` integration   |
| Hosting     | GitHub Pages          |
